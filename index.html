A11Y First - A Web Accessibility Workshop

Follow this workshop and see how we build a typical eCommerce browse and search page, learning about important accesibility concepts each step of the way.

Refer to the README for instructions.

Part 1 - Basic Concepts

Part 1 is primarily concerned with static text, page structure and links.

  1. Meta (the page will be intentionally blank)
  2. Paragraph
  3. Unordered List
  4. Heading
  5. Clipped Text
  6. Landmark
  7. Image
  8. Grid
  9. Link
  10. Card (presentational step)
  11. Tile
  12. Ambiguous Link
  13. New Window
  14. Faux Button
  15. Navigation Landmark
  16. Skip to Link
  17. Iframe
  18. Complementary Landmark

Part 2 - Intermediate Concepts

Part 2 is primarily concerned with buttons, form controls, DOM order and focus management.

  1. Carousel
  2. Placeholder Text
  3. Labels
  4. Form Validation (no JavaScript)
  5. Click Flyout
  6. Non Critical Icon
  7. Hover Flyout
  8. Critical Icon
  9. Access Key
  10. Tooltip
  11. Textbox
  12. Listbox
  13. Submit Button
  14. Search Landmark
  15. Faux Menu

Part 3 - Advanced Concepts

Part 3 is primarily concerned with advanced keyboard navigation, ARIA widgets & live regions.

  1. Menu
  2. Live Region
  3. Bubble Help
  4. Input Validation
  5. Form Validation (client-side)
  6. Dialog
  7. Tabs
  8. Combobox
